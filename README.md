# java-rmi-app
Java RMI app

**1. Make sure `javac` is on your path. On windows it's found here:**

`C:\Program Files\Java\jdk-11\bin`

**2. Compile java source files:**

`javac *.java`

**3. Start RMI service:**

`start rmiregistry`

**4. Start cruise control manager:**

`java CruiseControlManager`
